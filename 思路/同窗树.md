# 同窗树

- 产品概述：主要针对学生的的社交关系图谱

- 产品需求：
  - 陌生城市的孤独感成为弱社交关系的催化剂
  - 给废弃社交链一个重塑的机会
  - 基于共同场景基础的社交关系建立
  - 打破社交层级壁垒

- 产品功能：
  - 添加个人信息
    - 姓名
    - 头像
    - 学校
    - 签名
    - 联系方式
  - 隐私保护
    - 需要申请才能查看资料
    - 回答三到五个问题才能查看
  - 去中心化隐形人社交
    - 重新找到失联的隐形人
    - 没有发起人，每个人都能掌控自己的社交网
  - 基于地点的新型社交方式
    - 基于学校选课货或者活动或者地点的找人方式
    - 让你不再错失公开课上的女神
